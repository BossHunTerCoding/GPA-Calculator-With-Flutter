### วิธีคิดคำนวนเกรดเฉลี่ย (GPA)

นำเกรด (Grade) ที่ได้ของแต่ละวิชา (Course) มาคูณกับหน่วยกิต (Unit) ของวิชานั้นๆ<br>
แล้วนำมาบวกกันให้หมด จากนั้นหารด้วยผลรวมของหน่วยกิตทั้งหมด
